# Artistmatch-2.0

Premium Expo / React Native mobile app concept for artist networking with a map-first experience.

## Main Areas
- Home dashboard (project requests + chat previews)
- Interactive map with profile-image markers and mini profile preview
- Discovery feed with portfolio-first masonry cards
- Dedicated chat page
- Profile and full portfolio page

## Run
```bash
npm install
npm run start
```

## Expo SDK
- Expo SDK 52 (`expo ~52.0.38`)
- React Native `0.76.7`
- React Navigation v7
